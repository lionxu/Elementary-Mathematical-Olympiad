# 3-2-08-巧填算符综合

## 例题练习

### 例题1

在“+、-、×、÷、( )”中，挑出合适的符号，填入下面的数字之间，使算式成立。
$$
9\ \ \ 8\ \ \ 7\ \ \ 6\ \ \ 5\ \ \ 4\ \ \ 3\ \ \ 2\ \ \ 1=1
$$


### 练习1

在○内填上与等号左边不同的运算符号，使等式成立。
$$
(1)6-2+2=6\circ2\circ 2
$$

$$
(2)8+2+3=8\circ 2\circ 3
$$

$$
(3)16-8-3=16\circ 8\circ 3
$$



### 例题2

在下面算式的适当地方，填上运算符号“+、-、×、÷或( )”，使算式成立。
$$
1\ \ \ 2\ \ \ 3\ \ \ 4\ \ \ 5=10
$$


### 练习2

用运算符号把下面三个相同的数宇连接起来，使等式成立。
$$
(1)5\ \ \ 5\ \ \ 5=30
$$

$$
(2)6\ \ \ 6\ \ \ 6\ \ \ =30
$$
